# Command Block

The command block executes script commands through the engine script system.

## Behavior
- Opens its configuration UI via the block handler.
- Can be triggered by redstone systems and scripted events.

## Parameters
- `hardness`: Base mining toughness.
- `xp`: Metadata XP value.
- `color`: UI/editor tint hint.
- `emissive`: Enables glow treatment in materials.
- `emissiveColor`: Glow color used by the material registry.
