# AntonCraft Feature Backlog

## Core Gameplay
- Survival balancing pass (hunger, regen, enemy scaling)
- Crafting progression tiers (wood -> iron -> cyber tools)
- Dungeon / cave points-of-interest
- Virus corruption spread events

## World & Generation
- Biome variants (forest, dunes, frozen, corrupted)
- Deeper cave generation with rooms + tunnels
- Structure generation (ruins, camps, towers)
- Better chunk mesh optimization (greedy meshing)

## Combat & Mobs
- More mob AI archetypes (ranged, tank, support)
- Status effects (poison, slow, EMP)
- Boss encounters in corrupted zones

## UX / Polish
- Audio system (ambient + effects)
- Quest log and objective tracker
- Accessibility settings (crosshair, text size, color options)
- Controller support

## Performance
- Instanced block rendering by type
- Occlusion-aware chunk meshing
- Worker-thread chunk generation
- Runtime quality presets + auto-tune
